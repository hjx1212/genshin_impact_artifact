# genshin_impact_artifact
适用于hoshino v2的原神随机圣遗物插件

# 指令
- `@bot 圣遗物`: 获取一个随机圣遗物
- `@bot 圣遗物 [指定套装、部位、主属性、副属性]`: 获取一个指定的圣遗物，如：`@bot 圣遗物 魔女火伤杯副攻副攻副暴副暴`
